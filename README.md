Gesture-Controlled Media Interface

Arcane is a gesture-controlled media system that lets you control your computer using hand gestures through a webcam.

No keyboard. No mouse. Just your hands.

# What Arcane Can Do: 

* Play / Pause media

* Next / Previous track

* Control volume using hand movement

* Perform system actions using two-hand gesture combos

* Show a fullscreen futuristic HUD

* Includes a built-in gesture tutorial
